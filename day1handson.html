What is Css?

1.CSS stands for Cascading Style Sheets.

2.CSS describes how HTML elements are to be displayed on screen,paper,or in other media

3.Css is not necessary Css saves a lot of work. can control the lyout of multiple web pages all at once

4.External Stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline-by using the style attribute inside HTML elements
  • Internal-by using a style element in head section
  • External-by using a link element to link to an external css file
  • The most common way to add CSS is to keep the styles in external CSS files.However, in this tutorial we will use inline and internal styles,because this is easier to demonstrate,and easier for you to try it yourself

when comparing CSS class vs ID ,the difference is that CSS class apllies a style to multiple elemnts ,ID, on the other hand,applies a style to one unique ekemnt .ID is also special in that you can use a special URL to link directly to an element and it's used by javascript.

In CSS ,selectors are used to target a specific element or range of elements on a web page.Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors avaialble.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
